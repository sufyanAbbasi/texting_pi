Welcome to a message board community for Vassar by a Vassar student, since this website is only accessible within Vassar's network. To send a message to this board, send a text to +1(424)242-0412 in the format "message <your message>" If you text "help" to the number, you can find a list of other commands to use!
This project was created by Sufyan Abbasi and is running on a Raspberry Pi. The source code for the entire project is in this Github Repository: github.com/sufyanAbbasi/texting_pi.
Community Guidelines:
Use this as a space to express your thoughts about the world or to tell people that you exist. Your message is completely anonymous except for your designated color. Your phone number is never saved on the server (except for the last four digits for indexing purposes).
Please, no hateful speech meant to divide the Vassar community attacking race, gender, sexual orientation, religion, nationality, age or any other identity groups (or for trolling purposes). All comments will be left on the board to the best of my ability, so if you disagree with something, hide it by subtracting it away.
